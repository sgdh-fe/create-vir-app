## 安装

```sh
$ npm install -g create-vir-app
```

## 快速开始

```sh
$ vir test admin 0.1.1
$ cd test
$ npm install
# 开发模式
$ npm run dev
# 构建
$ npm run build
```

## 语法

```sh
vir <project-directory> [author] [version]

# [必填] project-directory  创建项目目录
# [选填] author             默认 git 的 username
# [选填] version            默认 1.0.0
```
